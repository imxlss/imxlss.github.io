# 任意属性

如果希望一个接口允许有任意的属性，可以使用如下方式：

interface Person {
  name: string;
  age?: number;
  [proName: string]: any;
}

let tom: Person = {
  name: 'Tom',
  gender: 'male'
};

# 只读属性

readonly定义只读属性。

TIP

只读的约束存在于第一次给对象赋值的时候，而不是第一次给只读属性赋值的时候。
