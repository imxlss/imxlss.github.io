# CSS 布局

# Viewport

视口在桌面端，指的是浏览器的可视区域

视口在移动端分为：

  • 布局视口(layout viewport)
  • 视觉视口(visual viewport)
  • 理想视口(ideal viewport)

# 布局视口

为了使为 PC 端设计的页面能在手机上正常浏览而产生。不与设备屏幕宽度不关联。会产生横向滚动条。

布局视口的尺寸

document.documentElement.clientWidth/Height

# 视觉视口

设备浏览器的可视区域。与缩放程度逆相关。

视觉视口的尺寸

window.innerWidth/Height

# 理想视口

对设备来说，理想的布局视口尺寸。

只有主动地往页面里添加meta视口标签时理想视口才会生效。

/* 这一行代码告诉浏览器，布局视口的宽度应该与理想视口的宽度一致 */
<meta name="viewport" content="width=device-width">
1
2

理想视口的尺寸

screen.width/height

# meta 视口标签

<meta name="viewport" content="name=value,name=value">
1
  • width：设置布局视口的宽度为特定的值
  • initial-scale：设置页面的初始缩放程度和布局视口的宽度
  • minimum-scale：设置了最小缩放程度(用户可缩小的程度)
  • maximum-scale：设置了最大缩放程度(用户可放大的程度)
  • user-scalable：是否阻止用户进行缩放
  • height：设置布局视口的高度(未被实现)

# 视口单位

视口单位有以下 4 个：

vw vh vmin vmax
视口宽度的 1% 视口高度的 1% vw 和 vh 中最小的那个 vw 和 vh 中最大的那个

# flex 布局

任何一个容器都可以指定为 Flex 布局。

.box {
  display: flex;
}
1
2
3

行内元素

.box {
  display: inline-flex;
}
1
2
3

Webkit 内核的浏览器，必须加上-webkit 前缀。

.box {
  display: -webkit-flex; /* Safari */
  display: flex;
}
1
2
3
4

TIP

采用 flex 布局的元素，称为容器。子元素自动成为容器成员，称为项目。

容器默认存在两根轴：水平的主轴（main axis）和垂直的交叉轴（cross axis）

主轴的开始位置（与边框的交叉点）叫做 main start，结束位置叫做 main end；

交叉轴的开始位置叫做 cross start，结束位置叫做 cross end。

# 容器属性

  • flex-direction

flex-direction 决定主轴方向

.box {
  flex-direction: row | row-reverse | column | column-reverse;
}
1
2
3
  • justify-content

justify-content 属性定义了项目在主轴上的对齐方式。

/* justify-content属性定义了项目在主轴上的对齐方式。 */

.box {
  justify-content: flex-start | flex-end | center | space-between | space-around
    | space-evenly;
}

  • align-items

align-items 属性定义项目在交叉轴上如何对齐。

.box {
  align-items: flex-start | flex-end | center | baseline | stretch;
}

  • align-content

当存在换行时，align-content 属性定义 行 在交叉轴上如何分配空间

.box {
  align-content: flex-start | flex-end | center | stretch | space-around |
    space-evenly;

# flex 项属性 (flex items)

  • order

order 属性可以控制它们在 flex 容器中的显示顺序。

.item {
  order: <integer>; /* 默认值是 0 */
}
1
2
3
  • flex-grow

flex-grow定义了 flex 项在有可用剩余空间时拉伸比例

.item {
  flex-grow: <number>; /* default 0 */
}
1
2
3
  • flex-shrink

flex-shrink定义了 flex 项的收缩的能力。(与 flex-grow 正相反)

.item {
  flex-shrink: <number>; /* default 1 */
}
1
2
3
  • flex-basis

flex-basis 定义了在分配剩余空间之前 flex 项默认的大小

.item {
  flex-basis: <length> | auto; /* default auto */
}
1
2
3
  • align-self

align-self 属性允许某个单独的 flex 项覆盖默认的对齐方式

.item {
  align-self: auto | flex-start | flex-end | center | baseline | stretch;
}
1
2
3

# Grid 布局

与 flexbox 的一维布局系统不同，CSS Grid 布局是一个二维布局系统

将容器元素定义为一个 grid 容器：

.box {
  display: flex;
}
