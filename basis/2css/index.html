# CSS 介绍

CSS指的是层叠样式表（Cascading Style Sheet），用来为网页添加样式。

# 基本语法

# 盒模型

CSS 盒模型有两种，可通过box-sizing属性来定义

  • content-box —— 标准盒模型（默认）
  • border-box —— 怪异盒模型
盒模型/属性 width height
标准盒模型 内容宽度 内容高度
怪异盒模型 内容宽度 + padding + border 内容宽度 + padding + border

# 选择器

常用选择器：ID选择器，类选择器，标签选择器

选择器 示例
伪类选择器 :first-child
属性选择器 a[title]
相邻兄弟选择器 img + p
通用兄弟选择器 p ~ span

# 选择器权重

权重决定了哪一条规则会被浏览器应用在元素上。

每种选择器都有自己的权重

选择器 权重
!important 1.0.0.0.0
inline-style 1.0.0.0
id选择器 0.1.0.0
类选择器 0.0.1.0
伪类选择器 0.0.1.0
属性选择器 0.0.1.0
元素选择器 0.0.0.1
伪元素选择器 0.0.0.1
通配选择器 0.0.0.0
  • 权重相同时，与元素距离近的选择器生效
  • 如果两个相同权重的选择器作用在同一元素上：以后面出现的选择器（样式中）为最后规则.

# scrollbar

::-webkit-scrollbar 仅仅在支持 WebKit 的浏览器 (例如, 谷歌 Chrome, 苹果 Safari)可以使用.

# 认识滚动条

::-webkit-scrollbar — 整个滚动条.
::-webkit-scrollbar-button — 滚动条上的按钮 (上下箭头).
::-webkit-scrollbar-thumb — 滚动条上的滚动滑块.
::-webkit-scrollbar-track — 滚动条轨道.
::-webkit-scrollbar-track-piece — 滚动条没有滑块的轨道部分.
::-webkit-scrollbar-corner — 当同时有垂直滚动条和水平滚动条时交汇的部分.
::-webkit-resizer — 某些元素的corner部分的部分样式(例:textarea的可拖动按钮).

# demo

/*定义滚动条高宽及背景
 高宽分别对应横竖滚动条的尺寸*/
::-webkit-scrollbar {
  width: 16px;
  height: 16px;
  background-color: #f5f5f5;
}
/*定义滚动条轨道
 内阴影+圆角*/
::-webkit-scrollbar-track {
  -webkit-box-shadow: inset 0 0 6px rgba(0, 0, 0, 0.3);
  border-radius: 10px;
  background-color: #f5f5f5;
}
/*定义滑块
 内阴影+圆角*/
::-webkit-scrollbar-thumb {
  border-radius: 10px;
  -webkit-box-shadow: inset 0 0 6px rgba(0, 0, 0, 0.3);
  background-color: #555;
}
