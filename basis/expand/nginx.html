# Nginx

# Nginx 是什么

免费、开源、高性能的 Http 服务器和反向代理，以及 IMAP/POP3 服务器。

反向代理

代理就是在 client 和 server 之间增加一层提供特定服务的服务器，即代理服务器。

正向代理

TIP

正向代理代理客户端，反向代理代理服务端。

1. 正向代理代理的对象是客户端，反向代理代理的对象是服务端
2. 正向代理隐藏了真实的请求客户端    // 代理客户端去访问服务器。
3. 反向代理隐藏了真实的服务端   // 帮我们把请求转发到真实的服务器

  1. 安装工具包 wget、vim 和 gcc
1. yum install -y wget
2. yum install -y vim-enhanced
3. yum install -y make cmake gcc gcc-c++

wget http://nginx.org/download/nginx-1.14.0.tar.gz

1. yum install -y pcre pcre-devel
2. yum install -y zlib zlib-devel
3. yum install -y openssl openssl-devel

tar -zxvf nginx-1.14.0.tar.gz -C /usr/local/

进入nginx-1.14.0目录然后在执行./configure命令

[root@xxx nginx-1.14.0]# ./configure --prefix=/usr/local/nginx

nobody   24957 24956  0 19:41 ?        00:00:00 nginx: worker process

[root@xxx nginx-1.14.0]#

# Nginx 环境变量

vim /etc/profile    (vim进入配置文件)

文件最底部添加：
    export NGINX_HOME=/usr/local/nginx(nginx安装路径)
    export PATH=$PATH:$NGINX_HOME/sbin
:wq 保存退出

source /etc/profile (使配置文件生效)

# 常见的环境变量

　　PATH：决定了shell将到哪些目录中寻找命令或程序

　　HOME：当前用户主目录

　　MAIL：是指当前用户的邮件存放目录。

　　SHELL：是指当前用户用的是哪种Shell。

　　HISTSIZE：是指保存历史命令记录的条数。

　　LOGNAME：是指当前用户的登录名。

　　HOSTNAME：是指主机的名称，许多应用程序如果要用到主机名的话，通常是从这个环境变量中来取得的。

　　LANG/LANGUGE：是和语言相关的环境变量，使用多种语言的用户可以修改此环境变量。

vim /usr/local/nginx/conf/nginx.conf
1

worker_processes  1;

#每个进程最大连接数（最大连接=连接数x进程数）每个worker允许同时产生多少个链接，默认1024
events {
    worker_connections  1024;
}
