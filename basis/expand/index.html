    # Unicode 和 UTF

    TIP

    简单说，Unicode是字符集，UTF-8是编码规则。

    Unicode是一套包含世界上所有字符的字符集，它为每一个字符分配一个码位。

    UTF-8是一套以8位为一个编码单位的可变长编码。会将一个码位编码为1~4个字节。

    # Unicode 总空间

    Unicode目前规划的总空间是17个平面（平面 0 ~ 平面 16），0x0000 至 0x10FFFF。每个平面有65536个码位。

    0x0000 至 0xFFFF 为 0 平面（Basic Multilingual Plane，BMP）。

    • BMP 的字符以UTF-16编码时使用 2 字节，以UTF-8编码时使用 1 至 3 字节。
    • 超出 BMP 的字符，以UTF-16或UTF-8编码时都需要 4 字节。
    • UTF-32编码任何字符都需要 4 个字节。
