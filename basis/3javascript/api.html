# Fetch API

Fetch API提供来一个全局fetch()方法和三个接口——Headers,Request,Response。

一个基本的 fetch 请求：

fetch('http://example.com/movies.json')
  .then(function(response) {
    return response.json();
  })
  .then(function(myJson) {
    console.log(myJson);
  });

# 页面生命周期

HTML 页面的生命周期有以下三个重要事件：

  • DOMContentLoaded：浏览器已经完全加载了 HTML，DOM 树已经构建完毕。但是像<img>和样式表等外部资源可能并没完成
  • load：浏览器已经加载了所有的资源（图像，样式表等）。
  • beforeunload/unload：当用户离开页面的时候触发。
DOMContentLoaded load beforeunload/unload
由 document 对象触发 由 window 对象触发 由 window 对象触发
HTML 已经完全加载，DOM 树构建完毕 浏览器已经加载了所有的资源（图像，样式表等 离开页面时触发
JS 可以访问所有 DOM 节点 获得图像的大小 询问用户是否保存了更改以及是否确定要离开页面

# onbeforeunload

当窗口即将被卸载（关闭）时，会触发改事件。

// 示例
window.onbeforeunload = function(e) {
  e = e || window.event;

  // 兼容IE8和Firefox 4之前的版本
  if (e) {
    e.returnValue = '关闭提示';
  }

  // Chrome, Safari, Firefox 4+, Opera 12+ , IE 9+
  return '关闭提示';
};

# Content-Type 和 MIME 类型

首部字段Content-Type用来说明实主体内对象的媒体类型。

媒体类型是一种标准，用来表示文档、文件或字节流的性质和格式。

MIME 的通用结构有主类型和子类型组成

TIP

type/subtype

类型 描述 常用
text 文本 text/plain, text/html, text/css, text/javascript
image 图像 image/gif,image/png,image/jpeg
audio 音频 audio/midi, audio/mpeg, audio/webm
video 视频 video/webm, video/ogg
application 二进制数据 application/octet-stream, application/json

上传文件时常用的multipart/form-data, multipart/byteranges

# target 和 currentTarget

Event 接口的只读属性 currentTarget 表示的是，当事件沿着 DOM 触发时事件的当前目标。它总是指向事件绑定的元素，而 Event.target 则是事件触发的元素。

例：

# history

DOMwindow对象通过history对象提供了对浏览器的会话历史的访问。允许你在用户浏览历史中向前和向后跳转。同时从 HTML5 开始——提供了对 history 栈中内容的操作。

向前和向后跳转：

和用户点击浏览器回退前进按钮的效果相同。

// 向后
window.history.back();
1
2
// 向前
window.history.forward();
1
2

跳到 history 中指定的一个点：

// 向后移动一个页面（等同于调用back()）
window.history.go(-1);
1
2
// 向前移动一个页面（等同于调用forward()）
window.history.go(1);
1
2

查看历史堆栈中页面的数量：

window.history.length;
1

# notification

Notification API的通知接口用于向用户配置和显示桌面通知。

构造方法

let notification = new Notification(title, options);
1

title —— 通知的标题

options 可设置的参数(常用):

  • body —— 通知的文本内容

  • icon —— 通知的图标图片的 URL 地址

  • data —— 和通知相关联的数据。

  • tag —— 通知的 ID

  • requireInteraction —— 通知是否保持有效不自动关闭，默认为false。

静态方法

  • Notification.requestPermission() —— 向用户请求通知显示权限

实例属性

常用属性：(readonly)

  • Notification.title

  • Notification.body

  • Notification.icon

  • Notification.data

  • Notification.tag

实例方法

  • Notification.close() —— 关闭通知

参考：notification

# window.requestAnimationFrame

window.requestAnimationFrame() 告诉浏览器——你希望执行一个动画，并且要求浏览器在下次重绘之前调用指定的回调函数更新动画。

语法

window.requestAnimationFrame(callback);
1

TIP

回调函数执行次数通常与浏览器屏幕刷新次数相匹配
