# 模块化

# require 和 import

两者遵循的模块化规范不同：

  • require/exports 野生规范
  • import/export ES 标准规范

由于import/export诞生的晚。目前编写的 import/export 最终都是编译为 require/exports 来执行的。

# 各自的写法

require/exports：

const fs = require('fs');

module.exports = fs;
exports.fs = fs;

import/export:

import fs from 'fs'
import {default as fs} from 'fs'
import * as fs from 'fs'

import {readFile as read} from 'fs'
import fs, {readFile} from 'fs'

export default fs
export const fs
export function readFile
export {readFile, read}
export * from 'fs'
