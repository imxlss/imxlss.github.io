# 一句话知识点

# target 和 currentTarget

Event.currentTarget 指向事件绑定的元素；Event.target 则是事件触发的元素。

# RxJS 的 Subject

调用next(value), Subject就会将值多播给订阅了它的观察者们。

# Angular 依赖注入

在 非懒加载模块 中申明的任何 provider 可以在程序内的任何地方被访问。

# Virtual DOM

用JavaScript模拟的 DOM 结构，将 DOM 变化的对比放在JavaScript层来做（diff）。

# TCP FLAGS 字段

SYN 表示建立连接，FIN 表示关闭连接，ACK 表示响应，PSH 表示有 DATA 数据传输，RST表示连接重置

# Promise 和 callback

Promise 和 callback 的本质区别是信任和控制（trust and control）。

# Map

Map 在涉及频繁增删键值对的场景下会有些性能优势

# 伪类和伪元素

伪元素是容器，伪类是选择器。伪元素为::；伪类为:。伪元素不会出现在 DOM 树中。
