# koa-body 中间件

koa-body是一个用来解析请求体的中间件。支持multipart, urlencoded 和 JSON请求体。

# koa-body获取参数

  • multipart表单
ctx.request.files; // 文件参数
ctx.request.body; // 非文件参数
1
2
  • 普通表单
ctx.request.body; // POST表单
ctx.query; // GET表单

# 配置参数

  • formidable 配置参数
参数名 描述 类型 默认值
keepExtensions 保留原来的文件后缀 Boolean false
multiples 是否支持多文件上传 Boolean true
onFileBegin 文件开始上传的回调 Boolean true
uploadDir 文件上传的目录 String os.tmpDir()

参考

# 遇到的坑

  1. 不要和 koa-bodyparser 同时使用

  2. 文件的获取

  • v4版本：ctx.request.files
  • v4之前版本: ctx.request.body.files

# 使用范例

// app.js
app.use(koaBody({
  multipart: true,
  formidable: {
    keepExtensions: true, // 是否包含拓展名
    maxFileSize: 200 * 1024 * 1024, // 上传文件大小限制
    onFileBegin(name, file) {
      file.path = `xxx` // 重命名上传文件
    },
    uploadDir: `xxx` // 上传文件保存的目录
  },
}));
